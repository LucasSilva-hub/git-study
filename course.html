课程基本情况

课程的性质和目的

     《HTML 程序设计》是计算机类专业的基础理论课，是从事基于 WEB 信息管理等应用系统开发 中实现 WEB 形式的 UI（用户交互界面）设计不可缺少的重要基础性设计工具。 通过本课程的学习，要使学生获得有关 WEB 设计及 WEB 内容前端处理，包括基于 WEB 的系统的 工作原理、WEB 界面的布局、前台的动态设计、后台的动态设计、WEB 系统的规划和设计等方面 的基本知识，掌握必要的基础理论和应用与设计方法及技巧。培养学生的应用 WEB 技术和工具设 计开发基于 WEB 的信息管理系统的技能，从而使学生初步具有运用 WEB 技术设计应用系统 UI（用 户交互界面）的能力，为专业后继应用系统开发类课程奠定必要的设计基础。

课程教学的基本要求

     1．本课程以课堂讲授为主，重在应用设计。在课堂教学中可适当结合知识单元的相对简单的 应用系统的设计进行教学，注重应用，开阔学生的视野，拓宽知识面。在作业和练习方面，任课教 师可以有针对生产实际适量地增加一定量的应用设计实训项目，以锻炼学生应用 WEB 技术设计解 决实际问题的能力。

    2．根据教育发展趋势和教学改革的要求，在本课程的教学过程中，应逐步引入多样化化教学 手段，如项目实训，在“做中学”。

    3．除教材外，应给学生指定相关的参考书，以拓宽学生的知识面。

    4．建议本课程每学期中间安排一次期中考试，期末考试实行教考分离。
